# 💰 Smart Budget

Intelligent expense tracker with AI categorization and spending insights

## Features

- 🧠 AI-Powered Analysis with MiMo model integration
- ⚡ Lightning-fast processing pipeline
- 🔒 Enterprise-grade security
- 📊 Real-time monitoring dashboard
- 🔄 Seamless CI/CD integration

## Tech Stack

- Next.js 14 / React 18
- TailwindCSS
- MiMo AI API
- Vercel Edge Functions

## Getting Started

```bash
npx create-next-app smart-budget-tracker
cd smart-budget-tracker
npm run dev
```

## Deployment

Deployed on [Vercel](https://vercel.com) with automatic CI/CD.
